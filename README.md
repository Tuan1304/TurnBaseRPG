# TurnBaseRPG

Game RPG Turn-Based được phát triển bằng Unity.

## Công nghệ sử dụng
- Unity 6
- C#
- FSM AI
- A* Pathfinding

## Chức năng
- Hệ thống chiến đấu theo lượt
- Nâng cấp tướng
- Trang bị
- AI quái vật

## Cách chạy
1. Mở project bằng Unity Hub
2. Chọn phiên bản Unity phù hợp
3. Nhấn Play
